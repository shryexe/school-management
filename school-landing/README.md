# EduFlow - School Management Landing Page

A modern, responsive landing page for a school management platform built with React and Tailwind CSS v4.

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS v4
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── Navbar.tsx      # Navigation bar
│   ├── Hero.tsx        # Hero section with CTAs
│   ├── Dashboard.tsx   # Dashboard mockup
│   └── LogosRow.tsx    # Trusted institutions logos
├── App.tsx
├── main.tsx
└── index.css           # Tailwind config & custom theme
```
